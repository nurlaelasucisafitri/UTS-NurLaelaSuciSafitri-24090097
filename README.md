## Deskripsi Singkat Project
Proyek ini adalah implementasi website sederhana (Single Page Application - SPA) yang dikerjakan sebagai Ujian Tengah Semester mata kuliah Pemrograman Web 1. Proyek ini dibangun menggunakan HTML, CSS, dan JavaScript dasar tanpa koneksi ke *backend* (server).

### Fungsi Tiap Halaman:
1.  **Halaman Login (`index.html`)**: Melakukan validasi sederhana bahwa email dan NIM tidak boleh kosong. Jika berhasil, akan diarahkan ke Halaman Dashboard.
2.  **Halaman Dashboard (`dashboard.html`)**: Menampilkan ringkasan data produk (Total Products, Total Sales, Total Revenue) dan memiliki tombol navigasi ke halaman List Data Produk.
3.  **Halaman List Data Produk (`products.html`)**: Menampilkan data produk dalam bentuk tabel. Terdapat fitur **Edit** (menggunakan alert dan *in-place editing* untuk bonus) dan **Delete** (menggunakan konfirmasi dan manipulasi DOM).
